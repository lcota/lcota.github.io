Trial 1
==================================================
First trial is using the following parameters for identifying regimes: 
## CPI 
HIGH: CPI YoY > 3.0
MED:  1.0 <= CPI YoY <= 3.0 
LOW:  CPI YoY < 1.0 

## NMI 
HIGH: NMI > 75th Percentile 
MED:  [25th Pctl <= NMI <= 75th Pctl]
LOW:  NMI < 25th Pctl 

## PMI 
HIGH: PMI > 75th Percentile 
MED:  [25th Pctl <= PMI <= 75th Pctl]
LOW:  PMI < 25th Pctl
